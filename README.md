
# EXPENSE TRACKER

Build an expense tracker as a react refresher


## Tech Stack

React

  
## Authors

- [@musebe](https://github.com/musebe)

  
## Run Locally

Clone the project


Go to the project directory

```bash
  cd expense-tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
